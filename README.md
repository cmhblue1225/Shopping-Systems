# 🛒 Shopping Systems - 통합 관리 시스템 모음

쇼핑몰 운영에 필요한 5가지 핵심 업무 시스템을 통합한 완전 오프라인 지원 웹 기반 솔루션입니다.

## 🌟 주요 특징

- **🌐 완전 오프라인**: 인터넷 연결 없이 모든 기능 사용 가능
- **📱 반응형 디자인**: 데스크톱, 태블릿, 모바일 모든 기기 지원
- **✍️ 디지털 서명**: 터치/마우스 기반 전자 서명 기능
- **📊 엑셀 출력**: 모든 데이터를 Excel 파일로 출력 가능
- **🔍 고급 검색**: 실시간 자동완성 및 다중 필터링
- **💾 로컬 저장**: 브라우저 로컬스토리지를 활용한 데이터 보존
- **🎯 즉시 사용**: 별도 설치나 설정 없이 바로 사용 가능

## 🗂️ 포함된 시스템

### 1. 📦 [입고요청서시스템](./입고요청서시스템/)
**물품 입고 요청 및 관리 시스템**

- ✅ 품목별 입고 요청서 작성 (최대 10개 품목)
- ✅ 자동 금액 계산 및 합계 표시
- ✅ 거래처 정보 관리
- ✅ 참조부서 다중 관리
- ✅ 3단계 승인 프로세스 (요청자 → 검토자 → 승인자)
- ✅ 다중 저장/불러오기 시스템

**주요 활용 분야**: 재고 관리, 구매 요청, 물류 관리

### 2. 👥 [회사원등록시스템](./회사원등록시스템/)
**직원 정보 등록 및 인사 관리 시스템**

- ✅ 포괄적 직원 정보 관리 (개인정보, 학력, 경력, 가족)
- ✅ 증명사진 업로드 및 관리
- ✅ 부서별, 직급별 분류 시스템
- ✅ 고급 검색 및 필터링 (입사일, 부서, 직급별)
- ✅ 인사카드 형태 출력
- ✅ 더미 데이터 자동 생성 (30명)

**주요 활용 분야**: 인사 관리, 직원 데이터베이스, 조직 관리

### 3. 📋 [물품신규등록시스템](./물품신규등록시스템/)
**상품 및 물품 등록 관리 시스템**

- ✅ 다양한 물품 카테고리 관리
- ✅ 상품 이미지 업로드 및 관리
- ✅ 가격, 재고, 공급업체 정보 통합 관리
- ✅ 바코드/QR코드 생성 및 관리
- ✅ 카테고리별 검색 및 필터링
- ✅ 재고 현황 및 통계 정보

**주요 활용 분야**: 상품 관리, 재고 관리, 카탈로그 관리

### 4. 🏢 [창고등록시스템](./창고등록시스템/)
**창고 정보 등록 및 관리 시스템**

- ✅ 최대 8개 창고 동시 등록
- ✅ 창고 유형별 분류 (일반, 물류, 냉장, 보세, 자동화)
- ✅ 지역별 창고 관리 (전국 8개 권역)
- ✅ 담당자 연락처 및 이메일 관리
- ✅ 창고별 상세 정보 (주소, 우편번호, 상세주소)
- ✅ 통합 검색 및 조회 시스템

**주요 활용 분야**: 물류 관리, 창고 운영, 배송 관리

### 5. 🤝 [거래처등록시스템](./거래처등록시스템/)
**거래처 정보 등록 및 계약 관리 시스템**

- ✅ 포괄적 거래처 정보 관리 (사업자등록번호, 대표자, 업태)
- ✅ 계약 조건 관리 (거래형태, 정산주기, 수수료율)
- ✅ 정산 정보 관리 (은행계좌, 세금계산서)
- ✅ 담당자 및 연락처 정보
- ✅ 출고지/반품지 주소 관리
- ✅ 계약 승인 프로세스 (MD → 팀장 → 부서장)

**주요 활용 분야**: 거래처 관리, 계약 관리, B2B 관계 관리

## 🚀 빠른 시작

### 1단계: 다운로드
```bash
git clone https://github.com/cmhblue1225/Shopping-Systems.git
cd Shopping-Systems
```

### 2단계: 시스템 선택 및 실행
각 시스템 폴더로 이동하여 `index.html` 파일을 브라우저에서 열기:

```bash
# 예시: 입고요청서시스템 실행
cd 입고요청서시스템
open index.html  # macOS
# 또는 더블클릭으로 브라우저에서 열기
```

### 3단계: 즉시 사용
- 인터넷 연결 불필요
- 별도 설치나 설정 없음
- 모든 기능 바로 사용 가능

## 🛠️ 기술 스택

### Frontend
- **HTML5**: 시맨틱 마크업 및 구조화
- **CSS3**: 모던 스타일링 및 반응형 디자인
- **JavaScript (ES6+)**: 동적 기능 및 인터랙션
- **jQuery 3.6.0**: DOM 조작 및 이벤트 처리

### Libraries (모든 라이브러리 로컬 포함)
- **jSignature 2.1.2**: 디지털 서명 캡처
- **ExcelJS**: Excel 파일 생성 및 출력
- **FileSaver.js 2.0.5**: 파일 다운로드
- **Cropper.js**: 이미지 크롭 및 편집 (일부 시스템)

### Storage
- **LocalStorage**: 브라우저 로컬 데이터 저장
- **JSON**: 데이터 직렬화 및 저장 형식

### 자주 묻는 질문

#### Q: 인터넷 연결 없이 정말 모든 기능이 작동하나요?
A: 네, 모든 라이브러리가 로컬에 포함되어 있어 완전 오프라인 환경에서 동작합니다.

#### Q: 데이터는 어떻게 저장되나요?
A: 브라우저의 LocalStorage를 사용하여 로컬에 저장되며, 브라우저 종료 후에도 보존됩니다.

#### Q: 모바일에서도 사용할 수 있나요?
A: 네, 반응형 디자인으로 제작되어 모바일 기기에서도 모든 기능을 사용할 수 있습니다.

#### Q: 엑셀 파일은 어떤 형식인가요?
A: .xlsx 형식으로 생성되며, Microsoft Excel, Google Sheets, LibreOffice에서 열 수 있습니다.
